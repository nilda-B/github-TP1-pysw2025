¿Qué es Angular?

Angular es un framework para construir aplicaciones web de una sola página (SPA) utilizando HTML y TypeScript.

Ventajas de Angular

Desarrollo rápido y fácil.
Soporte para la inyección de dependencias.
Gran comunidad y recursos.

Comunidad de Angular

Angular cuenta con una comunidad activa que contribuye con bibliotecas, herramientas y soporte.
